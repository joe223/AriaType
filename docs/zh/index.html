为什么不一样

面向桌面，而不是单个输入框

AriaType 关注当前应用、当前输入框和当前光标，让语音成为桌面的一部分，而不是另一个需要搬运内容的工具。

文字直达光标

说完就回到当前应用，不需要额外窗口，也不需要复制粘贴。

隐私默认开启

日常语音内容默认优先留在自己的设备上。远端服务可选，并且由你主动开启。
